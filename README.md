# Tema SuperGIROS Norte del Valle

Este es un tema personalizado de WordPress diseñado para la empresa **SuperGIROS Norte del Valle**. Está creado para ofrecer una experiencia web adaptada a las necesidades del negocio, proporcionando una interfaz intuitiva, amigable y profesional.

## Características del Tema

- Diseño personalizado para el negocio **SuperGIROS Norte del Valle**.
- Totalmente compatible con dispositivos móviles (responsive).
- Panel de Personalización de WordPress para ajustes rápidos y sencillos.
- Soporte para entradas, páginas y custom post types.
- Opciones de personalización del logo, colores y tipografía a través del Personalizador de WordPress.
- Widgets personalizados para mostrar información relevante, como promociones, ubicaciones y más.

## Requisitos

- **WordPress** 5.0 o superior.
- **PHP** 7.4 o superior.
- **MySQL** 5.6 o superior (o MariaDB).

## Instalación

### Opción 1: Instalación desde el panel de administración de WordPress

1. Ve a **Apariencia > Temas** en el panel de administración de WordPress.
2. Haz clic en el botón **Añadir nuevo**.
3. Haz clic en **Subir tema**.
4. Selecciona el archivo ZIP del tema (por ejemplo, `supergiros-norte-del-valle.zip`).
5. Haz clic en **Instalar ahora** y luego en **Activar**.

### Opción 2: Instalación manual (a través de FTP)

1. Descarga el archivo ZIP del tema.
2. Extrae el archivo en tu computadora.
3. Usa un cliente FTP (como **FileZilla**) para subir la carpeta del tema a la carpeta `/wp-content/themes/` de tu instalación de WordPress.
4. Ve a **Apariencia > Temas** en el panel de administración de WordPress y activa el tema.

## Configuración

Una vez que el tema esté activado, puedes personalizarlo a través del **Personalizador de WordPress**.

1. Ve a **Apariencia > Personalizar**.
2. Personaliza las opciones disponibles, como:
   - **Identidad del sitio**: Subir el logo de **SuperGIROS Norte del Valle**.
   - **Colores**: Cambia los colores del tema según la identidad visual de la marca.
   - **Carrusel de Inicio**: Sube las imagenes que deseas mostrar en el carrusel de imagenes de la página portada del sitio.
   - **Redes Sociales**: Ingresa la URL de las redes sociales de la empresa para el componente de redes sociales.
   - **Anuncios**: Establece la imagen y URL de distintos anuncios que se encontrarán en varias secciones de la página.
   - **Pie de página**: Cambia la información que se encuentra en el pie de página de una manera sencilla.

### Opciones adicionales

- **Menús**: Ve a **Apariencia > Menús** para configurar los menús de navegación publicos, privados y del pie de página.
- **Contenido estático**: Agrega páginas estáticas como "Nosotros", "Servicios", "Contáctanos", entre otras.

## Personalización de CSS

Si deseas realizar personalizaciones adicionales en el diseño, puedes agregar CSS personalizado directamente desde el Personalizador de WordPress:

1. Ve a **Apariencia > Personalizar > CSS adicional**.
2. Añade tu código CSS personalizado para ajustar el diseño a tus necesidades.

Ejemplo de CSS personalizado:

```css
/* Cambiar color de fondo en la cabecera */
.header {
  background-color: #003366; /* Azul de SuperGIROS */
}

/* Cambiar tamaño del texto en el pie de página */
.footer .text {
  font-size: 14px;
}
```
